# 🤖 Ivan Castro | Robotics & Automation Engineer Portfolio

Welcome to my professional portfolio repository. This project was engineered to showcase my expertise in industrial automation, robotics, and software development, bridging the gap between hardware logic and modern web technologies.

## 🚀 Tech Stack

* **Frontend:** React.js + Vite
* **Animations:** Framer Motion (for high-performance UI transitions)
* **Styling:** Tailwind CSS v4 (Modern Utility-First CSS)
* **Internationalization:** i18next (Full support for EN, PT, DE, FR)
* **Icons:** Lucide React

## ⚙️ Key Engineering Features

* **Industrial Preloader:** A custom boot sequence simulating a system initialization, featuring a dynamic Status LED (Red/Green logic) and multi-language technical greetings.
* **Multi-language Architecture:** Fully localized for the European market (English, Portuguese, German, and French), demonstrating readiness for global industry leaders like Bosch and Preh.
* **Modular Architecture:** Clean, component-based structure inspired by PLC modular programming (Function Blocks).
* **Responsive HMI Design:** A seamless User Interface (UI) that adapts to any screen size, mirroring the reliability of modern Human-Machine Interfaces.
* **Dark/Light Mode:** Integrated theme management with local storage persistence.
